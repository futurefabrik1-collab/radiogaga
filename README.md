# Ai Radio

An AI radio station running 24/7 with entirely AI generated content, music, dialogue, advertising, news, literature and interviews. There should be a web portal but also means to distribute the stream live globally continuiously. We will explore all open-source options and keep costs to absoliute minimum. Quality of ai radio stream will be improved through training, interactions with the audience and prompts.
